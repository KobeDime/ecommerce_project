**Kobe Essence
Welcome to Kobe Essence, a modern eCommerce application built with PHP and Laravel 10. 
This project is designed to provide a seamless shopping experience for users and a 
comprehensive management interface for administrators.

*Features

**User Features

Dashboard: Overview of user account details and recent activity.
Home: Main landing page showcasing featured products and promotions.
Shop: Browse and search for products, view details, and add to cart.
Why Us: Information about the benefits and unique selling points of Kobe Essence.
Testimonials: Read reviews and feedback from other customers.
Contact Us: Form for users to reach out with questions or comments.
Log In: Access user account by entering credentials.
Register: Create a new user account.

**Admin Features

Dashboard: Monitor total customers, products, orders, and delivered items.
Category Management: Add, edit, delete, and view product categories.
Product Management: Add, edit, delete, and view products.
Order Management: Add, edit, delete, view orders, change order status, and print PDF invoices.

**Installed Packages

Breeze: For authentication scaffolding.
Flasher: For flash messages and notifications.
Symfony: Various Symfony components for robust functionality.
Toastr: For beautiful, responsive notifications.
Barryvdh: Laravel DomPDF for PDF generation.
Stripe: Payment gateway integration for handling payments.

**Usage

*Access the User Dashboard:
Log in with user credentials and navigate through the various user features.

*Access the Admin Dashboard:
Log in with admin credentials to manage categories, products, orders, and track overall performance.
